# ✨ Overview

**Price List Table Manager** is a professional WordPress plugin designed for companies, agencies, educational institutions, and businesses that need a smart and modern pricing table management system.

The plugin allows administrators to upload Excel or CSV files and automatically generate responsive, searchable, and downloadable price tables for the frontend.

# 🔧 Installation Guide

## Manual Installation

1. Download the plugin ZIP file
2. Go to WordPress Admin Dashboard
3. Navigate to:

```bash
Plugins → Add New → Upload Plugin
```

4. Upload the ZIP file
5. Click **Install Now**
6. Activate the plugin
